# StellarPulse Chat Bot 使用说明

## 快速测试

```bash
# 查看帮助
cd ~/stellarpulse && python3 chat_bot.py "/help"

# 查看AI资讯
cd ~/stellarpulse && python3 chat_bot.py "/ai"

# 查看机器人资讯
cd ~/stellarpulse && python3 chat_bot.py "/robot"

# 搜索
cd ~/stellarpulse && python3 chat_bot.py "/search GPT-5"

# 查看详情 (回复列表中的数字)
cd ~/stellarpulse && python3 chat_bot.py "1"
```

## 支持的命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `/help` | 显示帮助 | `/help` |
| `/ai` | AI & 大模型资讯 | `/ai` |
| `/robot` | 具身智能 & 机器人 | `/robot` |
| `/space` | 航天 & 太空 | `/space` |
| `/tech` | 全部科技资讯 | `/tech` |
| `/hot` | 热门 TOP 5 | `/hot` |
| `/latest` | 最新 5 条 | `/latest` |
| `/search 词` | 关键词搜索 | `/search GPT-5` |
| `1-9` | 查看详情 | `3` |

## 消息自动响应

发送以下关键词，机器人自动回复：
- AI相关: `ai`, `/ai`, `人工智能`
- 机器人: `robot`, `/robot`, `机器人`, `具身智能`
- 航天: `space`, `/space`, `航天`
- 其他: `/tech`, `/hot`, `/latest`, `/help`
